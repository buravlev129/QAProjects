# Тестирование страницы "Конструктор календаря мероприятий"

## Описание проекта

Набор автотестов для проверки функциональности и валидации данных виджета "**Конструктор календаря мероприятий**".  
Адрес страницы `https://dev.3snet.info/eventswidget/`.

Тесты охватывают:
*   Положительные сценарии работы виджета
*   Анализ граничных значений для полей ввода
*   Проверку ввода невалидных данных в поля
*   Проверку корректности навигационных ссылок в главном меню

## Структура тестов

Тесты разделены на четыре категории.

### 1. Функциональное тестирование виджета

Проверка основного позитивного сценария работы виджета с различными комбинациями параметров.


| Название теста | Тип кейса | Описание | Параметры |
| :--- | :--- | :--- | :--- |
| **`test_calendar_random_valid`** | Валидный | Проверяет работу виджета<br> со случайными валидными<br> параметрами. | - **Тематика:** 3 случайных значения из списка.<br>- **Страна:** 3 случайных значения из списка.<br>- **Ширина/Высота превью:** случайное значение из валидного диапазона |
| **`test_calendar_list_select_all_valid`** | Валидный | Проверяет работу виджета<br> при выборе всех доступных<br> опций в списках | - **Тематика:** "Выбрать все".<br>- **Страна:** "Выбрать все".<br>- **Ширина/Высота превью:** случайное значение из валидного диапазона |


### 2. Тестирование граничных значений для полей "Ширина" и "Высота"

Проверка корректной обработки валидных и невалидных числовых значений.

| Название теста | Проверяемое поле | Проверяемые значения |
| :--- | :--- | :--- |
| **`test_width_field_accepts_boundary_values`** | **Ширина, px** | Граничные значения: `[239, 240, 1020, 1021]` |
| **`test_height_field_accepts_boundary_values`** | **Высота, px** | Граничные значения: `[229, 230, 750, 751]` |


### 3. Тестирование ввода невалидных значений в поля "Ширина" и "Высота"

Проверка корректной обработки ввода невалидных значений.  
Для проерок использовались: `0`, `-1`, `ааа`, `230px`, `Пустая строка`, `Пробелы`, `Большое число`

| Название теста | Проверяемое поле |
| :--- | :--- |
| **`test_width_field_rejects_invalid_values`** | **Ширина, px** |
| **`test_height_field_rejects_invalid_values`** | **Высота, px** |


### 4. Проверка ссылок главного меню

| Название теста | Описание | Критерий успеха |
| :--- | :--- | :--- |
| **`test_header_navigation_links`** | Проверяет доступность всех ссылок в <br>главном меню страницы. | 1. Собирает все ссылки из меню.<br>2. Открывает каждую в отдельном окне/вкладке.<br>3. **Тест считается пройденным,** если отсутствуют ссылки<br> с ошибками при открытии страницы  (например, `404`) |


## Возможные направления для дальнейшей работы

*   Добавление тестов на производительность
*   Проверка корректности отображения данных в виджете (даты, тематика, страна)
*   Тестирование поведения виджета при изменении размеров окна браузера
